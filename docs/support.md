# Support Gina

Gina is free, open-source software. It is developed and maintained by [gina-io](https://github.com/gina-io), a small web development company. Donations and community contributions help fund continued development, documentation, and support.

---

## Sponsor on GitHub

GitHub Sponsors support is being set up — the sponsor page is not yet live. Check back soon.

Once available, recurring sponsorships of any amount will be welcome. Even $5/month makes a meaningful difference for a small open-source project.

---

## Star the repository

A GitHub star costs nothing but has real impact — it improves discoverability, signals activity to potential users, and helps Gina reach package indices that require a minimum star count.

[Star gina-io/gina on GitHub →](https://github.com/gina-io/gina)

---

## Contribute

Code contributions are the highest-leverage form of support. The project actively welcomes pull requests for:

- Bug fixes
- Documentation improvements
- New connectors (MySQL, PostgreSQL, ScyllaDB — see [Roadmap](/roadmap))
- Test coverage

[See the contributing guide →](https://github.com/gina-io/gina/blob/master/CONTRIBUTING.md)

---

## Spread the word

- Share Gina with other Node.js developers
- Write a blog post or tutorial
- Answer questions in [GitHub Discussions](https://github.com/gina-io/gina/discussions)
- Report bugs and suggest improvements in [GitHub Issues](https://github.com/gina-io/gina/issues)
